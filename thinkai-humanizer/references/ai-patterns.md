# 中文文案 AI 痕迹参考

只处理文案中实际存在的问题。以下模式是诊断线索，不是见词必删的黑名单。

## 结构痕迹

| 痕迹 | 常见示例 | 处理方向 |
|---|---|---|
| 标准背景开场 | 随着时代发展、在当今社会 | 直接进入问题、场景或结论 |
| 工整递进 | 首先、其次、最后 | 保留内容，删除序列外壳或重排 |
| 二元反转 | 看似……实则…… | 各自独立表达，删除人为反转 |
| 固定升级 | 不仅……更……、不只是……更是…… | 拆成两句或删掉前半个框架 |
| 空洞提示 | 值得注意的是、需要指出的是 | 直接说后面的事情 |
| 标准总结 | 总的来说、综上所述 | 删除套话，必要时保留具体结论 |
| 强行升华 | 这不仅是产品，更是一种生活方式 | 删除无依据升华，停在具体价值上 |
| 段落同构 | 每段都是观点—解释—总结 | 调整段落功能和节奏 |

## 用词痕迹

| 类型 | 常见表达 | 处理方向 |
|---|---|---|
| 空泛最高级 | 极致、非凡、卓越、惊艳 | 删除或用具体场景、效果替代 |
| 夸张变革 | 颠覆、革命性、彻底改变、重新定义 | 说明具体解决了什么问题 |
| 商务黑话 | 赋能、加持、焕新、升级体验 | 改成用户可以做什么、少了什么麻烦 |
| 无效确定 | 毋庸置疑、不可否认、毫无疑问 | 删除，直接陈述事实或判断 |
| 过量程度 | 实在是太、真的非常、极其 | 有依据则保留必要程度，无依据则精简 |
| 伪社交语气 | 宝子们、姐妹们、绝绝子、yyds | 仅在原作者真实语气中保留 |
| 过度号召 | 闭眼入、无脑冲、必入 | 改成适用条件或明确建议 |

## 句式痕迹

| 痕迹 | 处理方向 |
|---|---|
| 连续多句以同一主语开头 | 换主语、调整语序或合并 |
| 连续反问 | 保留最有必要的一句，其余改陈述 |
| 相邻句长度和结构高度一致 | 混合长短句和独立判断句 |
| 破折号、冒号使用过密 | 用句号或自然衔接拆开 |
| 一段反复使用“但、然而、同时” | 删除无效转折，合并相关信息 |
| 每句话都承担完整论证 | 允许有的句子只给判断或事实 |
| 固定三选一 CTA | 直接建议、自然结束或提出真实问题 |

## 内容痕迹

- 同一观点换词重复两到三遍。
- 为了显得全面，把所有可能性都罗列一遍。
- 每个优点后都补一句没有新信息的解释。
- 只有“方便、高效、体验好”，没有具体动作和结果。
- 用“很多人、大家都、用户普遍认为”代替证据。
- 为保持中立而不回答“值不值、适不适合”。
- 结尾重复全文而没有新增信息。

处理时优先删除无效信息，不要压缩掉事实、条件、风险和必要解释。

## 伪人类痕迹

以下做法不能真正提升自然度：

- 随机加入“说实话、讲真、就是说”。
- 每段都用“我”制造亲身感。
- 添加没有依据的试用经历。
- 故意使用错别字、病句和异常标点。
- 强行加入流行梗和网络热词。
- 把正式作者统一改成短视频博主。
- 把所有句子切成同样短的碎片。

自然表达来自具体信息、真实取舍和符合作者身份的节奏，不来自表演式口语。

## 判断原则

遇到疑似 AI 表达时依次判断：

1. 这句话是否包含不可替代的信息？
2. 问题来自内容本身，还是只来自表达外壳？
3. 删除后是否损失事实、条件、立场或逻辑？
4. 能否用更具体、更直接的方式表达？
5. 改写后是否仍像原作者，而不是另一种固定人格？
