# 事实来源规则

## 证据标签

- `[FACT]`：客户材料或可靠来源明确支持。
- `[INFERENCE]`：基于事实的合理推导，必须保留推导边界。
- `[ASSUMPTION]`：工作流暂时假设，等待确认。
- `[UNKNOWN]`：没有足够信息。

## 来源优先级

1. 客户确认的产品资料、包装、说明书、合同或官方页面。
2. 品牌官方公开资料。
3. 可核实的权威第三方资料。
4. 客户提供的竞品内容，仅用于结构与市场表达分析。
5. 社交内容，只能作为线索，不能单独证明产品效果和数据。

事实性声称必须记录来源、查询日期和支持范围。核心事实为 `[ASSUMPTION]` 或 `[UNKNOWN]` 时阻断交付。
