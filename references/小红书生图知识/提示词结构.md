# 简洁提示词结构

Prompt 的目标是给模型清楚边界，而不是替模型完成全部视觉设计。

只保留：

1. 页面要表达的单一任务。
2. 品牌和产品名称。
3. 仅首图写“参考图片主体为品牌产品，参考官网图片”。
4. 必须出现的中文文字。
5. 字体的大致气质。
6. 3:4 小红书成品图。

默认不写：

- 逐项摄影参数。
- 复杂镜头和道具清单。
- 过细的材质、灯光、背景和配色控制。
- 长负面词列表。
- 本地合成、抠图、叠加或后期排版要求。

首图推荐句式：

```text
参考图片主体为{品牌和产品}，参考官网图片，其余构图和场景自由发挥。为小红书制作一张 3:4 成品图，表达{页面任务}，自然呈现中文“{精确文字}”，字体气质为{字体气质}。
```

第二页起不再引用官网图：

```text
为小红书制作一张 3:4 成品图，表达{页面任务}，自然呈现中文“{精确文字}”，字体气质为{字体气质}。其余视觉创意、构图和场景自由发挥。
```

能删则删。只要页面任务、产品身份、参考图、文字和比例没有丢失，就把创意空间交给模型。

批次规则：每条 Prompt 的最终文本必须不同，不允许在同一批次中重复使用 100% 完全相同的 Prompt。不做生成后图片相似度自检。
