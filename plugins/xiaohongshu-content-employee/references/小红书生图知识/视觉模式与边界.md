# 视觉执行边界

- 每个安装环境首次使用时列出全部可用图片渠道和具体模型，由用户选择并保存默认项。
- 选题确定后，全部最终 Prompt 直接并发发送给已保存的默认模型。
- Prompt 固定写明 `3:4` 小红书成品图，请求使用所选渠道的默认竖版尺寸。
- 需要保持产品外观时，只给首图绑定清晰的官方产品参考图。
- 模型原图直接进入 HTML，不用代码加字、裁切、抠图、叠图或重新合成。
- 图片返回后不验图、不截图、不复审 Prompt、不重生成，立即进入 HTML 生成。
- 用户明确要求更换模型时才重新选择；普通任务不重复询问渠道或逐条确认 Prompt。
- 图片能力不可用时，允许回退到真实图片与信息卡，不影响文案交付。
